# StudentHub – Week 2 Dashboard

StudentHub is a student productivity dashboard built as part of a Full Stack Web Development Bootcamp.

## Features

### Task Manager

* Add new tasks
* Mark tasks as completed
* Delete individual tasks
* Clear all tasks
* Task completion statistics
* Data persistence using Local Storage

### Habit Tracker

* Track daily habits such as:

  * Drink Water
  * Study
  * Exercise
  * Read
  * Sleep on Time
* Reset habits with a single click
